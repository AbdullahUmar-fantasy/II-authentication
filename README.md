## Quick Start
Install:

* NodeJS 14.*
* Internet Computer dfx CLI

Clone the Git repository:  
```bash
git clone https://github.com/dappblock/nextjs-ic-starter
```

Enter the commands:
```bash
cd nextjs-ic-starter
npm install
dfx start
dfx deploy
npm run dev
```

Open Chrome:  
http://localhost:3000/
